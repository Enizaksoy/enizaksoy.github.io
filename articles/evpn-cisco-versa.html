Overview

This project demonstrates EVPN-VXLAN implementation across two datacenters. The primary datacenter uses Cisco Nexus infrastructure, while the secondary datacenter utilizes Versa Flex VNF. Both datacenters are interconnected through an MPLS backbone network.

Key Features

  • Multi-datacenter VXLAN EVPN fabric with BGP as control plane
  • L2/L3 VNI extension between Cisco infrastructure and Versa Flex VNF
  • VRF segmentation and tenant isolation across datacenters
  • BGP EVPN Type-2/Type-5 routes for host/prefix advertisement
  • Anycast gateway implementation for optimal traffic forwarding
  • L3VNI (VLAN 999) for inter-VRF communication
  • LACP-based port-channels for link redundancy

Use Cases

  • Datacenter Migration: Move workloads between physical and virtual infrastructure
  • Hybrid Cloud: Connect on-prem DC with SD-WAN enabled branches
  • Disaster Recovery: Active-active or active-standby DC configurations
  • Multi-site Deployment: Consistent networking across heterogeneous platforms
