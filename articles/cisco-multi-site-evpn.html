Overview

A complete BGP EVPN VXLAN multi-site data center fabric built on Cisco CML, using AI-assisted configuration. The lab implements a production-style 5-stage Clos topology with two PODs interconnected through a Super-Spine layer — 20 network devices in total (14 NX-OS 9000v, 6 IOSvL2 access switches), each running its own BGP AS for optimal multipath.

Key Features

  • eBGP EVPN overlay with a unique ASN per device for optimal multipath
  • Anycast gateway — same IP (192.168.x.1) and MAC on every leaf
  • VRF mylab with L3VNI 50000 for inter-VLAN routing
  • Ingress replication (BGP-based) for BUM traffic — no multicast required
  • Multi-site design — POD isolation with Super-Spine interconnect
